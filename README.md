# 🍽️ Ethiopian Feast  
### **Compare Dishes • Macro Tracker • Cultural Food Insights**

Ethiopian Feast is a beautifully designed, interactive web app that helps users explore Ethiopian cuisine through **nutritional insights**, **dish comparisons**, and **macro‑focused analysis**.  
Built with a clean, cinematic UI, the app makes it easy to understand how traditional dishes differ in calories, protein, carbs, and fats — all while celebrating the richness of Ethiopian food culture.

---

## ✨ Features

### 🔍 **Dish Search & Exploration**
- Browse authentic Ethiopian dishes with detailed macro breakdowns  
- View calories, protein, carbs, fats, and serving‑size information  
- Clean, modern UI optimized for clarity and speed  

### ⚖️ **Side‑by‑Side Comparison**
- Compare two dishes instantly  
- Visual macro comparison for quick decision‑making  
- Perfect for meal planning and nutrition tracking  

### 📊 **Macro‑Focused Insights**
- Understand which dishes are higher in protein, lighter in calories, or more balanced  
- Great for fitness, weight management, and cultural food education  

### 🎨 **Premium UI/UX**
- Minimal, elegant layout  
- Smooth interactions  
- Mobile‑first responsive design
